# Kotlin Inventory Management System

## Overview
This is a console-based application designed to help manage stock for a small business. It allows users to track items by a unique ID, name, quantity, and price.

I developed this software to master the fundamentals of the Kotlin programming language, specifically focusing on data structures and user input handling.

## Development Environment
* **Language:** Kotlin 1.9.0+
* **IDE:** IntelliJ IDEA
* **Build System:** Gradle / Kotlin Compiler

## Execution Instructions
1. Open the project folder in **IntelliJ IDEA**.
2. Run the `Main.kt` file by clicking the green arrow next to the `main()` function.
3. Use the numeric menu (1-6) to interact with the inventory database.
